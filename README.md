# 艾尔登法环 剧情解析

世界背景
==============================================================================

在过去，褪色者跟随王离开交界地的时候，据说有一艘船留了下来（锈蚀船锚）。

主线（第一人称）
------
- 褪色者开始于侯王礼拜堂，旁边有个死去的指头女巫，地上写着**即使法环早已破碎，也请您当上艾尔登之王**，接下来出门需要面对一个接肢贵族，并且可以看到附近都是几个相对来说比较华贵的墓地。之后出现在漂流墓地，被梅琳娜看到，实际上是被托雷特捡到，**所以托雷特很可能不是一匹简单的灵马**。之后出门在引导之始见到梵雷。**可以看到梅琳娜和梵雷等都在英雄墓地等候，应该在等新的褪色者到来**。梵雷他说我很惨什么都没有（没有指头女巫，不能获取卢恩力量，不能去圆桌厅堂），但是可以看到赐福指引，应该跟着指引去史东薇尔。在艾雷教堂遇到圣诞老人，流浪商人咖列。然后往北经过关卡前方。关卡前方会遇到裁缝柏克。他被欺负之后裁缝工具被抢走了，然后在海岸洞窟打败可以夺回来还给他。通过关卡前方往史东薇尔走会遇见小红帽，她说她的同伴都被干了变成接肢的怪物，自己很害怕。往东北走能见到肯尼斯海德，帮他夺回要塞之后，他和亚人合作，并且希望出现一个新的王在宁姆格福。

地区故事
==============================================================================

法姆亚兹拉
------
+ 有只法姆亚兹拉的兽人在宁姆格福的近林洞窟，里面很多狗，旁边有史东薇尔士兵的尸体，**看着像是兽人在此有很久了，士兵查过来之后被弄死了**

宁姆格福
--------
+ 飞龙亚基尔在宁姆格福湖里，但其实初见他是飞到这里来的，不知道是一直在宁姆格福还是刚飞过来。击败得到龙心脏。
+ 傍海古遗迹下面有一只白王，能拿到战灰-重力
+ 画家的破屋，拿到画-归巢
+ 肯尼斯海德的那个要塞里夺回之前躺着一个亚人女王尸体，要塞外面的灵魂叙述有提到到。附近还有很多血蔷薇。外面有个黄金树种子。
+ 肯尼斯海德的那个要塞里能拿到迪可达斯符节（左）。
+ 王侯断崖，玩家的指头女巫：即使引导早已破碎，也请您当上艾尔登之王。
+ 很多魔像残骸的附近捡到骑枪护符
+ 斗技场附近，被“叛律者”亨利克斯入侵，击败得到大槌护符。

啜泣半岛
--------
+ 监视塔左边那片林子里面，有个位置有个小帐篷，那种很小的三角帐篷叶子搭的，旁边死了一个人，有两个混种埋伏，那个死的人很有可能是商人。
+ 巡礼教堂里面是拉达冈的雕像，玛莉卡箴言说葛孚雷众战士阿返回交界地吧使劲干仗吧死了之后会复活的。
+ 岛中间有个癫火村
+ 封印监牢里是萨米尔古英雄，打完掉一个拉达冈的烙印
+ 灵庙平原的灵庙有钟
+ 归还塔，直接传送王城的神授桥，而神授桥是去玛莲妮亚神授塔的位置
+ 摩恩城BOSS狮子混种，可以召唤城主艾德格帮助。狮子混种身后有个石碑，中间有个洞的，洞周围有指纹，**应该是英雄墓碑，摩恩城亡国英雄的墓碑**。
+ 林子里有亚人女王，击败之后得到亚人女王杖。附近的地窖里捡到罪人盾

灵庙
--------
+ 灵庙原野的灵庙附近有小白花，五瓣或者六瓣。
+ 灵庙原野的灵庙附近有灵庙守卫，无头。

大赐福
--------
+ 第一次去，柯林做了自我介绍，狄亚罗斯要找勒妮亚，铁匠说自己就喜欢打铁，菲雅要抱抱，恩夏（百智爵士的保镖）站在书房门口（获得动作：有何贵干），百智爵士在阴阳怪气。
+ 拿到圣树秘密符节之后，回到大赐福会被“王骸”恩夏入侵。然后在书房门口能拿到她的王骸一套。
+ 一楼厨房，拿到克雷普的黑键，再往里拿到刺客祷告书。

史东薇尔城
--------
+ 进城BOSS恶兆妖鬼马尔基特，可以召唤魔法师罗杰尔。
+ 城墙塔电梯下面的接肢，旁边房间有一个很大的尸体堆，也有一只倒吊的山妖，对应的就是很多褪色者被杀了作为接肢的祭品，小红帽的同伴也都死在了那里。
+ 城里面有一只被锯了角的恶兆。
+ 城里面能拿到绘画-预言
+ 打完葛瑞克，穿过王座往里走拿到夏波利利葡萄
+ 候王礼拜堂，四钟楼山传送过来，击败接肢贵族后裔，获得仪式直剑，兽纹黄金盾。里面拿到风暴鹰古王，风暴鹰汀涅。

湖之利耶尼亚
--------
+ 利耶尼亚大道最南边靠着史东薇尔的路头，马车里面是大树矛
+ 大道监视塔捡到卡利亚辉剑杖
+ 复仇者的破屋, 被发狂了的艾德格入侵，屋子里全是生肉丸，他身上掉落生肉丸，夏波利利的葡萄，失乡骑士戟+8
+ 四钟楼山脚附近，四钟楼山，有两个无头的山妖骑士，类似灵庙骑士那种状态，卡利亚装束。会用魔法。山顶第一个钟楼能捡到魔石剑钥匙。
+ 壶村，植物种类特别丰富
+ 白金村，BOSS是恶兆猎人，能捡到泪滴幼体，结晶剑。
+ 辉石龙史玛拉格，后面放着伤龙油脂，学院辉石钥匙。
+ 封印杜鹃的监牢里面是“卡利亚骑士”波尔斯，击败得到巨剑阵。
+ 有一个篝火叫：湖中坠落遗迹。**推测：所有地面上的巨大的遗迹建筑碎块都是从天上掉下来的，可能是法姆亚兹拉？**
+ 卡利亚书斋，有一个魔法教授米丽安，使用魔法罗蕾塔的大弓，击败掉落魔法天降魔力。
+ 米丽安的形象和塞尔维斯是一样的，**推测：塞尔维斯基本可以确定是个傀儡，原型可能就是米丽安。**
+ 利耶尼亚神授塔，顶部是菈妮的尸体，橘红色头发，长袍，领口有红宝石，左右手各有一个镶嵌红宝石的手镯。得到死亡咒痕（右）和观星少女的传说。
+ 卡利亚城寨，BOSS是禁卫骑士罗蕾塔（虚影），击败得到罗蕾塔的大弓，战灰罗蕾塔的斩击。
+ 卡利亚城寨，毕迪，好像在偷偷摸摸地干什么，能买到老旧地图，指引**琥珀色光芒**
+ 塞尔维斯有个放傀儡的秘密地下室，里面有信息：塞尔维斯的傀儡，请勿随意触碰。
+ 王室墓地的封印监牢，里面是石肤黑王，击败得到：法术陨石
+ 癫火村会被“烧灼指痕”维克入侵，击败后得到指痕葡萄，维克的战矛。
+ 癫火村，镇静教堂，有一个死去的指头女巫，拿到指头女巫一套。**猜测：按照灵魂的说法，这个维克本来是个褪色者，镇静教堂的指头女巫是他的女巫，可惜他没能成王。**
+ 通过矿道上到亚坛高原，途中有熔岩土龙马卡尔，击败得到土龙鳞剑和龙心脏。
+ 在菈妮剧情后拿到被丢弃的王室钥匙，可以打开女王旁边的箱子，获得暗月戒指。

魔法学院
------
+ 一个墓碑之前，跪着三个不死人，能捡到卡利亚骑士一套。
+ 女王大书库前面，“卡利亚骑士”穆格拉姆，击败得到卡利亚骑士盾。
+ 后面的洞窟击败结晶人，爬上来到最高处能捡到魔法之境。

盖利德
------
+ 熏火教堂，“吞噬褪色者”安娜塔西亚，会入侵，击败得到圣对蝎。
+ 盖尔要塞，拿到碎星的传说
+ 艾奥尼亚沼泽岸边，会被米莉森入侵，她会玛莲妮亚的剑舞，击败得到圣血木芽和卢恩。
+ 击败老将尼奥，得到纯净金针。
+ 魔法镇瑟利亚大门外面，格威写的留言：请到破屋一趟吧
+ 法洛斯要塞，捡到迪可达斯符节（右）
+ 野兽祭司偶尔会从神殿里出来，跑到神殿侧面，对着天空吼叫
+ 击败拉塔恩后，星星开始转动，一颗大流星落在宁姆格福，发出耀眼的白光。
+ 杰廉离开红狮子城之后，城里再度被占领，击败熔炉骑士后获得遗迹大剑。

地下
------
+ 希芙拉河，通过点燃火焰，可以去打祖灵
+ 希芙拉河，有一只龙人士兵，巨人龙头，肚子也被掏空，击败得到武器龙戟。附近能拿到玛莉卡的烙印。
+ 安瑟尔河下游，诺克史黛拉的龙人士兵，击败后获得奇迹冰雷枪，后面可以拿到一个大朵灵依墓地铃兰。
+ 诺克隆恩的那个大球球，周围有很多的银色泪滴空壳，**猜测大球球是很多泪滴的集合体**
+ 诺克隆恩，黑夜神域拿到猎杀指头刀，和大朵灵依墓地铃兰。
+ 安瑟尔河主流，捡到娇小菈妮。

火山官邸
------
+ 走廊了有个灵魂形态的人哼哼唧唧的：“呜。。。呜呜。。。有没有人啊？快杀了他吧。。。快杀了大蛇，快杀了拉卡德大人。。。呜。。。你是褪色者吧？是挑战半神的人吧？拜托，快杀了大蛇。快杀了吃掉拉卡德大人的大蛇。弑蛇的矛就放在谒见厅。用那把矛才能杀死大蛇。褪色者啊，快用那把矛刺穿大蛇吧。。。以前的拉卡德大人抱有亵渎的野心，那是霸王的雄心壮志。但是在大人被大蛇吃掉之后，那雄心壮志摇身一变，沦落成卑贱的贪欲。那东西已经不是拉卡德大人了。。。必须杀了那东西。不能再让那东西侮辱大人的名声与野心。”干掉拉卡德之后，可以在这个人影的位置捡到：致谢。
+ 格密尔火山附近有罗德尔骑士和士兵，已经疯癫，会癫火，有些趴在地上啃食尸体。
+ 格密尔火山，坠星成兽，击败后获得星兽半颚。
+ 火山官邸猎杀古老骑士伊修托邦，获得鳞片一套。
+ 火山官邸猎杀忒拉格斯，获得大山羊一套。
+ 火山官邸猎杀莱利，获得克雷普的小瓶子。
+ 击败神皮贵族，得到：神皮缝针，贵族气场。在祭坛上拿到：蛇的羊膜。
+ BOSS，一阶段大蛇，二阶段拉卡德：哦。。。还不坏啊。你啊。。。成为我蛇王的族人吧，一起吞噬神祇吧！**拉卡德的剑上面有很多的手臂，大蛇身上也有无数的像蛇信子一样的手伸出来，根据BOSS画面暗示，成为族人应该就是被大蛇吞掉，融为一体，变成力量之一。后面塔妮丝也有提到，吞食后成为拉卡德的蛇，成为族人。** 战败的话，拉卡德会说：好啦，你也一起吞噬神祇吧！击败后获得：拉卡德的大卢恩，亵渎君王的追忆，拉卡德会说：。。。我不受任何事物掌控，蛇永恒不灭。
+ 击败拉卡德之后，干掉塔妮丝，会被塔妮丝的骑士入侵，击败之后获得：熔炉百相之喉囊。塔妮丝原来的位置捡到：侧室一套。

亚坛高原
------
+ 毕迪处购买的老旧地图指向进王城大门之前的附近有琥珀星光，来到此出有很多圣血木芽，几个小章鱼，以及琥珀星光，有一个雕像，形象是一个女性跪地护住一个跪地的孩子。
+ 日荫城旁边的荒地上，很多雕像，有一个魔像，“日荫城主”玛雷玛雷在这里，击败后得到蚁刺细剑，玛雷家一套。
+ 日荫城能拿到，女武神的义手。击败BOSS“铁棘”艾隆梅尔之后，获得：玛雷家行刑剑，铁棘大盾。
+ 王城门口的封印监牢，接肢的葛孚亚，击败后得到葛孚雷的肖像。
+ 风车村的人，手持各式剥皮工具，身穿庆典礼服。村里很多T形的架子。BOSS神皮使徒，击败后得到：神皮剥制剑，横扫黑焰。**猜测：风车村和神皮制作有关系，庆典剥皮献祭神皮使徒制作神皮，剥制剑和灵魂叙述都有暗示。**
+ 蠕血废墟，可以入侵“兽爪”马格纳斯，击败后获得：巨星锤。
+ 亚坛高原中间盆地地区很多蚯蚓脸，小黄金树旁有一个蚯蚓脸boss。蚯蚓脸的攻击带有死亡属性。**猜测：蚯蚓暗示靠近黄金树根系，黄金树根系带有死亡属性。**
+ 王城外围可以遇到“古龙”兰斯桑克斯，击败后获得：兰斯桑克斯的刀。
+ 小黄金树教堂，里面有颗树，旁边很多黄金百足。
+ 王成外围，战场上有很多扎进地里面的大柱子，**应该是巨大的箭，首先都是向王城里面倾斜，然后每一支都炸出深坑，并且其中有几个能看到箭头的形状。**
+ 王成外围，有很多的英雄墓碑，符合剑冢的描述：第二次罗德尔保卫战很多英雄死在这里。

王城罗德尔
------
+ 进入王城之后，梅琳娜就说她可以自由活动了。
+ 王城的房屋，大部分都闭着门，而且从门缝中有黄色的流体溢出来。
+ 王城贫民窟的教堂，拿到：死眠连衣裙，莱恩尼尔一套。**猜测菲雅和莱恩尼尔之前生活在王城的这个教堂里**
+ 下水道击败“恶兆之子”蒙格，获得：血焰爪痕。
+ 下水道的墓地，击败“鲜血祭司”艾斯加，获得：鲜血君王的欢愉。
+ 王城圆桌遗址大厅，拿到阿尔佩利希一套。和贝纳尔一起入侵“离群魔法师”维赫勒，“白狼战鬼”巴格莱姆，击败后获得战鬼一套。
+ 黄金树大教堂，击败“初始之王”葛孚雷（的影子），得到护符皮袋。拿到：黄金律法原本。

祷告
===============================================================================

冰雷枪
------
- 能展现龙人士兵力量的祷告。
- 能召唤冰雷枪，从空中刺入地面。雷枪反弹后，会朝前方释放落雷。
- 龙人士兵诞生自永恒之城，他们没见过真正的天空与雷电，将冰雷作为自己的武器。

回归性原理
------
- 黄金律法基本主义的祷告。形成根基的祷告之一。
- 能治愈所有的异常状态，并消除特殊效果，去除所有拟态，揭露真面目。
- 基本主义用两种力量解释黄金律法---回归与因果。所谓的回归，意指将万物朝不变收敛，属于含义上的引力。

拉达冈的光环
------
- 黄金律法基本主义的其中一种祷告。父亲拉达冈赠予年幼米凯拉的回礼。
- 能形成，扩大黄金光环，攻击四周。借由蓄力能扩大攻击的范围。
- 然而年幼的米凯拉抛弃了基本主义---因为对于玛莲妮亚的宿疾，基本主义无能为力。此为“纯净无比的黄金”的起点。
