梅琳娜
------
- 捡到玩家：不用担心，托雷特，他还有救。这个人肯定也会追求法环，即使这么做会背离黄金律法。
- 对话，赐福1：穿越雾前来的人，我是梅琳娜，我想要跟你谈个条件，你知道指头女巫吗，他们侍奉双指，帮助，指引褪色者，但是现在的你，没有女巫，我可以代替他们的职务，我能将卢恩化为力量，只要你渴求老头环，这件事就对你有帮助，但是你要带我到黄金树脚。。。。这样就算谈成了。还有一件事，这枚戒指交给你，需要长距离移动，就用它吧，那能够呼唤名为托雷特的骏马的灵魂，托雷特选上了你，希望你好好待它。
- 对话，赐福2：这簇小巧的金光，是黄金树的赐福，据说过去，这金光从你们褪色者的眼眸散失，而如今，又转而出现，引导你们。你应该看的见吧，赐福的指引，引导使命所在的光芒。史东薇尔有一位碎片君王，他是在法环破碎之际，拥有破碎一部分的一个半神，如果赐福指引你前往那里，那你就真的收到法环的引导。对和你有着约定的人而言，我希望你能接受法环的引导，挑战各种难关。
- 对话，赐福3：关于我吗，我在寻找。过去母亲在黄金树赐予我的那个使命，还有即使肉身烧毁，变成灵魂形貌的身体，我还能继续活命的理由。。。。。其实我也想向你道歉，我曾说要代替指头女巫的职务，但是我没有办法引导你，因为我不是女巫，也失去了既有的使命。。。
- 对话，打完恶兆妖鬼：对不起，其实我一直在观察你，怀疑你是不是真的受赐福指引，还有，会不会挺身挑战难关，结果是我杞人忧天。从一开始，托雷特就相信你了，我不够真诚。有一件事情，是我能引导你的-圆桌厅堂，那是受到指引的褪色者，也就是英雄们聚集的地方。我能够带你前往（我要去圆桌厅堂）。我知道了，一下子就好，让我碰触你。

亚人柏克
-------
- 在宁姆格福，关卡前方的一片树丛。
- 对话，初次见面：对哦，我被施法变成树了，我被从洞窟赶出来，回不去了，还被变成树。对不起，我被赶出来时候，东西都忘带了，只能用这个感谢你（蘑菇）。再等我一下，我想偷偷回洞窟，那会东西，我就一定能帮你的忙。但是要再等一下，我还是很怕他们，需要做点心理准备，一想到他们，还有那个海岸的洞窟，我就全身都动不了。
- 对话，在海岸洞窟：你怎么来了，这里很危险，快点逃吧，他们肯定会跑来攻击你，最后落得和我一样的下场。
- 对话，给完缝衣针：这该不会是，缝衣针。谢谢你，我真的很高兴，我妈妈是裁缝，我想成为她那样的裁缝，我不可以这么轻易死掉。

血指猎人-尤拉
--------
- 对话，宁姆格福傍海古遗迹，初见：你是新来的褪色者吧，而且你还成功打到了龙，既然如此，该告诉你，你手上那颗龙心脏，是龙飨用的供品。等到某一天，你为了获得力量吃下它，心中因此萌生渴望，难以克制的时候，记得到西海岸的尽头，有座小岛，到岛上的废弃教堂一趟。不过，你也要记得，所有执行龙飨的人，都会渐渐成为非人，渴望龙的念头不会消失，而是会渐渐积累，最后那个念头充斥全身，不断折磨着你。龙的力量既强大又美丽，却十分可怕，也因此，执行龙飨等同踏上毁灭的不归路。
- 在山谷溪里会被血指涅利乌斯入侵，尤拉会来帮忙。
- 打完之后再沿着小溪往里走，能看到尤拉，对话：又见面了，和涅利乌斯交战又幸存，你确实有一点本事。老夫叫尤拉，是猎杀血指的猎人，他们是为咒血痴狂的褪色者，一群袭击同胞的恶徒。在这块土地上游走，肯定免不了与他们一战。听好了，就算你念在彼此是同胞，那群家伙也早就疯了，不可能和他们沟通，可别因为犹豫就手下留情。快离开吧，彼此有缘就会再相见。

瑟濂
--------
- 对话，初见，驿站街遗迹：我名为瑟濂，如你所见，是一名魔法师，那么有何贵干？（我想学习魔法），哦你想学习辉石魔法，但老实说，我不认为你能学好辉石魔法，学习魔法并不是只看资质，然而拜师要慎选对象，我被逐出雷亚卢卡利亚学院，是众人避之唯恐不及，所谓的异端魔女，即使如此，你还坚持拜师吗？（我还是想拜师），哈，你真是怪人，好吧，我愿意收你为徒，传授辉石魔法，但我很严格，不走温和教导那一套，可别后悔啊。
- 初次学习魔法，获得动作：点头称是
- 对话，拜师后：徒弟啊，第一步该做的是理解，你在魔法的领域还是个婴儿，极度不成熟，甚至连路都不会走，然而认知到自己的不成熟，才能成为魔法师。我们开始上课吧。
- 对话，拜师后：还想继续上课吗，我乐意奉陪。

肯尼斯海德
---------
- 对话，初次靠近：来人啊，有没有人在，任何能帮助我，肯尼斯海德的人啊，志愿摒除不合理，维护正当秩序的人啊。啊，黄金树，请带来您的侍者吧。
- 对话，初见：哦哦出现了，但是。。。不，即使是褪色者，也值得欢迎，不限身份尊贵与否，皆能维护正当秩序，我是肯尼斯，是宁姆格福领主的嫡长子，褪色者啊，我有事儿要你帮忙。从这里往南，穿过雾林有一个要塞，那是我的要塞，希望你帮我夺回来，都怪史东薇尔派来的骑士长，实在过于驽钝，竟然会为血痴狂。
- 对话，再次：你还在这里做什么，快点往南穿过雾林夺回我的要塞。啊，原来如此，你想问报酬吧，别小看我，打赏我绝不吝啬，只要确实夺回要塞，肯定大大有赏。啊，给你一个建议，最好小心葛瑞克的褪色者猎人，那群家伙一心想抓到你们褪色者，当做恐怖的接肢的祭品。只是凭葛瑞克那种格局，根本不配跻身君王之流。战败之后混在女人堆里逃出王城，忌惮拉塔恩，躲在城里一个气也不敢吭。小看玛莲妮亚，结果输的落花流水，还跪地亲吻她的脚表示服从。实在是软弱无比，不知羞耻的男人。即便出身名门一族，也看得出来葛孚雷的血统淡了很多。仔细想想还真是可悲。
- 对话，夺回要塞后：啊啊，是你啊，我等了好久了，你成功夺回我的要塞了？成功了吗！骑士长死了吗！做的好，做的非常好，我的眼光果然够精准，好，这是说好的报酬，大方收下吧（获得武器：黄铜短刀）。
- 对话，夺回要塞后再次：我准备要回去要塞了，该做的事情堆积如山，首先，必须要重启与亚人之间的交流。瞧你一脸狐疑，只要身在黄金树之下，即使是亚人也会是芳邻，“那些卑贱的，正当的秩序不会屏除在外”，这也是宁姆格福的正统政权后代，我肯尼斯应尽的使命。你就好好见证一切吧。
- 对话，夺回要塞后三次：啊，是你啊，我看时机也成熟了，有没有意思在我底下工作？我很看好你的能力，在我这里工作，学习所谓黄金树之下的正当秩序，说不定有一天，赐福也会重回你那对褪色的双眼。你怎么想？在我看来是个不错的提案（愿意），哦，明智的选择，你果然前途无量，我在要塞等你，届时封你为骑士。。多么美好的未来，名副其实的“黄金树赐福我等”。

小红帽，罗德莉卡
----------
- 对话，初见：大家都被接肢了，和我一起来到交界地，为我而战的那些人，手，脚，头全部被砍断，变成蜘蛛的一部分了，你知道吗，被接到蜘蛛身上之后，人就会长得像是虫蛹喔，真的很奇妙对不对（或者动作：收拢双腿而坐）。你一个人行动吗？朝着史东薇尔城前进吗？还是因为听信那位白面具先生的好听话？还是说，你想要变成蜘蛛的一部分呢？如果是的话，那就和我一样了。可是我没有勇气，要被砍断手脚，砍断头，太可怕了。我很想变得和大家一样，但是在怕的不得了，我太懦弱了。
- 对话，再次：啊对了，能请你带着这孩子一起旅行吗（获得：灵魂水母的骨灰），要它陪着这么懦弱的我，太可怜了，而且灵魂好像很喜欢你，我想这孩子也会很开心。
- 对话，三次：那么，请多保重，若果你准备前往史东薇尔城，请帮忙转达给成为虫蛹的大家，我爱你们，虽然还是很害怕，但迟早会变得和大家一模一样，我终于比较不怕痛了。
- 对话，找到蛹群的遗物，交出：这是什么。。。这是大家的遗物？啊。。。这是。。。竟然有这种事。。。大家相信我，即使现在什么事都办不到，还是相信在未来，我能够成就些什么。。。我想前往圆桌厅堂，去找出自己能做些什么。**猜想：这里应该是小红帽从物品上面的灵魂了解到了一些信息。**
- 坐火之后，小红帽离开了风暴破屋，留了一盏蜡烛，一束花。
- 大赐福，对话：好久不见，我叫罗德莉卡，抱歉，这么晚才告诉你名字。说起来，圆桌厅堂很厉害呢，即使双指的藏身处，也是想当上王的英雄群集的地方。。。真没想到褪色人的指引，背后竟然充满着奥秘。。。只是对我而言，圆桌厅堂的指引过于伟大。我想找找其他自己能做的事儿。
- 大赐福，再次对话：啊，还有一件事，请让我向你道谢。如果你没有将大家的遗物转交给我，我肯定会一直坐在原地，自怨自艾，继续背叛大家的期待。我很感谢你，请收下这个（获得：黄金种子）。
- 如果没把遗物给小红帽，她会自己去圆桌厅堂，黄金种子会留在暴风破屋。
- 对话，转达铁匠的话：我有那方面的潜力吗？真不敢置信，但如果是真的。。。如果是真的，就算我不够优秀，我也想去学习，想去磨炼。我想请修古先生指导我。那位先生外表恐怖，被束缚在圆桌厅堂，厌恶我们褪色者。会被讨厌是理所当然，但是。。。我确认为，那位先生其实非常温柔。
- 对话，成为调灵师之后：啊，谢谢你。你替我说服修古先生了，对吧？多亏如此，修古先生愿意教导我关于调灵的学问。我还不知道自己能走多远，但只要一想到自己能尽绵薄之力帮助你们，也能慰藉被接肢的大家，我就想继续努力下去。调灵实习生罗德莉卡，今后请多多指教。

铁匠
----------

涅斐丽
----------

圣诞老人
----------
对话：我们幸存的族人，应该也在这块土地上讨生活，要是你遇到他们，有兴趣的话，和他们买点东西吧。。。。我们流浪民族从古至今都不受黄金赐福的青睐。因此我们被迫四处为家，过着困顿的旅行生活。在艾尔登法环破碎之后，我们还能保留神志，可能这就是原因吧。光这点看来，褪色者和流浪民族，我们两个还颇类似的。
对话：嗯，这个警告或许对你没有意义，记住，绝对不要攻击我们的族人，流浪民族有一条族规，“孤单也很好，即使受人轻视，被人抛弃，也不必想着谋求什么，唯独那些伤害我们的，决不能轻易放过”。要说的话，这就是我们的律法。这民族听起来很执着，很烦人吧。
