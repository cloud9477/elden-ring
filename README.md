# 艾尔登法环 剧情解析

世界背景
=============

在过去，褪色者跟随王离开交界地的时候，据说有一艘船留了下来（锈蚀船锚）。

剧情梗概
=============

主线（第一人称）
-----
- 我开始于**fix**，旁边有个死去的指头女巫，地上写着**fix**，接下来被**打败**。之后出现在漂流墓地，被梅琳娜捡到，实际上是被托雷特捡到，所以托雷特很可能不是一匹简单的灵马。之后在引导之始见到梵雷，他说我很惨什么都没有（没有指头女巫，不能获取卢恩力量，不能去圆桌厅堂），但是可以看到赐福指引，应该跟着指引去史东薇尔。在艾雷教堂遇到圣诞老人，流浪商人咖列。然后往北经过关卡前方。关卡前方会遇到裁缝柏克。他被欺负之后裁缝工具被抢走了，然后在海岸洞窟打败可以夺回来还给他。通过关卡前方往史东薇尔走会遇见小红帽，她说她的同伴都被干了变成接肢的怪物，自己很害怕。往东北走能见到肯尼斯海德，帮他夺回要塞之后，他和亚人合作，并且希望出现一个新的王在宁姆格福。

人物故事
=============

梅琳娜
------
- 捡到玩家：不用担心，托雷特，他还有救。终于找到了。这个人肯定也会追求法环，即使这么做会背离黄金律法。
- 对话，赐福1：你好，穿越雾前来的人，我的名字是梅琳娜，我想要跟你谈个条件，你知道指头女巫吗，他们是侍奉双指，帮助，指引褪色者的人，但是现在的你，没有女巫在身边，我可以代替他们的职务，我能将卢恩化为力量，只要你渴求艾尔登法环，这件事就对你有帮助，所以，希望你能帮忙，带我到黄金树脚。。。。这样就算谈成了，想让卢恩化为自己的力量时，请在赐福所在之处呼唤我。对了，还有一件事，这枚戒指交给你，需要长距离移动时，就用它吧，那能够呼唤名为托雷特的骏马的灵魂，托雷特选上了你，希望你好好待它。
- 对话，赐福2：这簇小巧的金光，是黄金树的赐福，据说过去，这金光从你们褪色者的眼眸散失，而如今，又转而出现，引导你们。你应该看的见吧，赐福的指引，引导使命所在的光芒。史东薇尔有一位碎片君王，他是在法环破碎之际，拥有破碎一部分的一个半神，如果赐福指引你前往那里，那你就真的收到法环的引导。对和你有着约定的人而言，我希望你能接受法环的引导，挑战各种难关。
- 对话，赐福3：关于我吗，我在寻找。过去母亲在黄金树赐予我的那个使命，还有即使肉身烧毁，变成灵魂形貌的身体，我还能继续活命的理由。。。。。其实我也想向你道歉，我曾说要代替指头女巫的职务，但是我没有办法引导你，因为我不是女巫，也失去了既有的使命。。。
- 对话，打完恶兆妖鬼：对不起，其实我一直在观察你，怀疑你是不是真的受赐福指引，还有，会不会挺身挑战难关，结果是我杞人忧天。从一开始，托雷特就相信你了，我不够真诚。有一件事情，是我能引导你的-圆桌厅堂，那是受到指引的褪色者，也就是英雄们聚集的地方。我能够带你前往（我要去圆桌厅堂）。我知道了，一下子就好，让我碰触你。
- 对话，巡礼教堂：这儿地方存在箴言，是很久以前就销声匿迹的玛莉卡女王，她所留下的箴言。。。你想听的话，我可以转达。（回答：我想听）。。。我直接复诵玛莉卡的箴言：在你们面临死亡之后，那被夺去的会再归还---返回交界地，投身交战，随心所欲地展现艾尔登法环。与死亡比邻，成为强者吧---王的众战士啊，吾王葛孚雷啊。
- 对话，玛莉卡第三教堂：这个地方也有玛莉卡女王的箴言。你还想听的话，我可以转达。（回答：我想听）。。。我知道了。我直接复诵玛莉卡的箴言：吾王啊，王的众战士啊，我将夺去你们的赐福。当你们的双眸暗淡褪色，我将放逐你们于交界地之外---于交界地之外征战，生存，死亡吧。。。怎么样？希望多少能代替女巫的引导。

白面具，梵雷
------
- 初见，对话：哦，你是褪色者吧。着实可悲，没有女巫陪伴，没有引导，没有获取卢恩的力量，不能去圆桌厅堂，只能死的默默无名吧。即使你没有女巫，也有一丝希望，因为你遇到了我。你知道赐福吧，就是能让你们褪色者歇息的金光，正是赐福的指引，褪色者该行进的路，指向该去哪里，或者该葬身何处。我想赐福指的是断崖上的城，史东薇尔，又老又丑的半神，接肢葛瑞克的居所，
- 胡之利耶尼亚，蔷薇教堂，对话：噢，等你很久了。不仅成功取得了大卢恩，也觐见了圆桌厅堂的双指。那么，你有什么想法？（我觉得很感动）很好，那么请你出发吧，我的使命已经达成。愿你有双指的指引。（我觉得双指不对劲）你果然也有感觉啊。老实说，我也有同感。双指的话不能相信。仿佛是上了年纪的老人在说满口空话。我心里是这么想的，当艾尔登法环破碎之际，双指与它的指引是否也同样毁坏了？而且，我感受不到双指对我们的爱，这是我最不能忍受的部分。
- 怀疑双指后，再次对话：噢，还有一件事。有件物品想要送给聪颖的你（获得物品：溃烂血指）。这是可以摆脱双指指引的手段，请你务必一试。如果你对此感到满意，我们再见面吧。我很期待再会呢，亲爱的。
- 三次对话：送给你的手指，请务必一试。你根本不需要犹豫，赐福跟双指都曾对褪色者弃之不顾。。。等到面临毁灭了，又向你们求助。对于这种存在，哪有必要讲求情谊？

菈妮
------
- 对话，首次，艾蕾教堂：初次见面啊褪色者，我是魔女蕾娜，听说有褪色者骑着灵马，让我花了点时间去找，看来就是你啊，那名为托雷特的灵马，你能召唤它，没错吧？（回答：我能召唤灵马）。。。。嗯，答得好。我有物品要交给你，是托雷特前主人托给我的，（招魂铃，三匹狼），那物品叫作召魂铃，没有回归黄金树的骨灰，能从中召唤出灵魂，而那些灵魂会暂时认你为主人，重现生前战斗的模样。。。哎，照你的意思使用吧。褪色者啊，打扰你了。我想我们不会再见面了，仔细品尝交界地的滋味吧。。。。我很期待啊，你们褪色者会服从双指到什么时候。
- 对话，菈妮塔，没有从罗杰尔那里听到菈妮的信息：哦，好久不见了。记得当时我自称蕾娜，很高兴看到托雷特过得不错。那么，褪色者啊，你是为了什么而来？我应该没有发邀请函给你。（回答：没事）没事吗？这样啊。到有趣了，那我们在此重逢，可以说是命运从中牵线。你要替我做事吗？我是魔女菈妮。过去窃走死亡，如今找寻深入黑暗的路。总有一天，我会背叛一切，也会舍弃一切。怎么样，听起来有趣吧？（回答：愿意替魔女菈妮做事）这样啊，那很好，你真是奇人，那种邀请也肯答应。但我的门槛也不过如此。期待你往后的表现。
- 对话，菈妮塔，从罗杰尔那得到菈妮的信息：哦，好久不见了。记得当时我自称蕾娜，很高兴看到托雷特过得不错。那么，褪色者啊，你是为了什么而来？我应该没有发邀请函给你。（回答：魔女菈妮是阴谋之夜的主谋吧？）原来如此，调查得真仔细。没错，我是魔女菈妮。曾经窃走部分死亡卢恩，再经由仪式，化成弑神用的黑刀---我是做过这些事。再来，你想找的咒痕，不在这里。我杀害，抛弃与生俱来的身体---而咒痕就刻在那身体上。
- 对话，那副身体在哪里：为什么非得告诉你？我不隐瞒过去，不为此感到后悔，但也不会有人乞讨就乐于施舍。你想找的咒痕不在这里，我言尽于此。好了，尽快离开。
- 对话，我想要帮忙：哦？你的意思是，要替我做事？你认为这么做，就能找到咒痕的所在地？。。。也行，我不讨厌心怀鬼胎的人。况且，如果是以前的作为---那咒痕将你带来，这命运也颇有意思。我准许你替我做事。努力寻找咒痕的下落吧。
- 对话，菈妮塔：那么，立刻有事派给你。有一位名叫布莱泽的半狼战士，他也在替我做事。我要你和他联手，找出永恒之城诺克隆恩的秘密宝藏。我已经召来布莱泽，他人在楼下，详情去问他吧。对，还有军师尹吉，魔法教授塞尔维斯应该也在。他们颇为特立独行，有必要时善加利用吧。各取所需，他们肯定也不会客气。去和楼下的那三个人聊几句吧。不要求你融入，但是你免不了受他们的打量。毕竟很久没有新人加入了。
- 对话，菈妮塔：。。。啊，还有一件事。我就要进入深眠。短时间内不会醒来。人偶的身体也不如想象中的自由。。。我会期待---下一次睁眼的时候，能听见好消息。

魔法师，托普斯
--------
- 初见，对话：你是不是褪色者？既然是的话，方便分一些卢恩给我吗？俗话说人不可貌相，我曾待过雷亚卢卡利亚学院，在那学习辉石魔法。只要你分卢恩给我，我可以教你魔法（分给他10卢恩），哦哦，感谢啊，太感谢了。你真是好人。我叫托普斯。照我们说好的，如果你有兴趣学，我愿意教你魔法。只是我会的，也不是上得了台面的魔法。

圣杯露滴
------
- 各个教堂收集到
- 供奉在各处教堂内的黄金树恩惠。此为所剩无几的碎屑。能够提升圣杯瓶的回复量。

------
- 黄金树的神父---灵药师们留下的物品。结晶露滴的形成历时长久，蕴藏的力量能透过此物牵引而出。加入不同的结晶露滴调配，将产生各种特殊效用。在赐福处休息能获得补充。

缝衣针

- 亚人柏克很珍惜的缝衣针，弯如兽牙，粗大的缝衣针

罪人盾
------
- 在啜泣半岛的亚人森林捡到
- 女巫受罪恶荆棘刺穿双眼。为了祭祀女巫而制作的盾。据说女巫会在此地再次诞生。

剑骸大剑
------
- 在啜泣半岛的摩恩城击败BOSS拿到
- 摩恩城收藏的传说中的武器。肩负无数悲叹，愤怒的复仇大剑。**传说中的武器之一**
- 在过去，只身独活的亡国英雄，收集了一族战士所有的武器，继续战斗。

锈蚀船锚
------
- 在宁姆格福坑道击败BOSS挖石山妖得到
- 将生锈的船锚当做是武器挥舞的物品，粗厚尖锐的四支锚爪，让攻击带有突刺属性
- 在过去，褪色者跟随王离开交界地的时候，据说有一艘船留了下来

卡利亚辉剑杖
------
- 在利耶尼亚大道监视塔顶捡到

象牙小镰刀
------
- 以象牙加工制成的小镰刀。年迈的白金之子们的武器。
- 他们志愿侍奉不曾见过的圣树，据说此为侍奉的证明。

结晶剑
------
- 人力无法制作的纯粹结晶剑。带有魔力，会依照智力提升攻击力。
- 永远守护着结晶，充满谜团的结晶人。那背后的原因有一种说法：等到他们的创造者回来时，能请求创造者用结晶雕刻出新的同胞。

剑冢
=============

宁姆格福，关卡前方
------
- 黄金格瑞克，耻辱之战，交战米凯拉的锋刃，兵败如山倒。跪伏于地，恳求饶命。

啜泣半岛，摩恩城
------
- 摩恩攻城战。只身一人的复仇英雄挺身而战，最终败给葛孚雷王。

湖之利耶尼亚，大道旁
------
- 玛莲妮亚向南推进的纪念碑。“米凯拉的锋刃”以及尊腐骑士-那高展的飞翼，无人能敌。

结缘教堂旁边
------
- 第二次利耶尼亚战争。黄金与月，无人胜出。取而代之诞生的，是赎罪与结缘。

卡利亚城寨旁边
------
- 卡利亚城寨保卫战。卑鄙杜鹃尸横遍野之处。

宁姆格福和盖利德交界处
------
- 葛孚雷王的征战，最后的地点。黄金军队势如破竹，百战百胜---然而失去赐福，色彩褪去。

盖利德大道北方
------
- 艾奥尼亚战争，拉塔恩，玛莲妮亚战成平手，猩红腐败之花怒放。

盖利德不破大桥
------
- 碎星战争。拉塔恩化为瑟利亚之盾，独自一人挺身而出，击碎星星。

啜泣半岛，啜泣的封印监牢
------
- 萨米尔古英雄
- 掉落：拉达冈的烙印
