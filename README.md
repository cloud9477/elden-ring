人物故事
=============

梅琳娜
------
- 对话
 + 捡到玩家：不用担心，托雷特，他还有救。这个人肯定也会追求法环，即使这么做会背离黄金律法。
 + 赐福：穿越雾前来的人，我是梅琳娜，我想要跟你谈个条件，你知道指头女巫吗，他们侍奉双指，帮助，指引褪色者，但是现在的你，没有女巫，我可以代替他们的职务，我能将卢恩化为力量，只要你渴求老头环，这件事就对你有帮助，但是你要带我到黄金树脚。。。。这样就算谈成了。还有一件事，这枚戒指交给你，需要长距离移动，就用它吧，那能够呼唤名为托雷特的骏马的灵魂，托雷特选上了你，希望你好好待它。

白面具，梵雷
------
- 对话
 + 首次遇到玩家：着实可悲，没有女巫陪伴，没有引导，没有获取卢恩的力量，不能去圆桌厅堂。即使你没有女巫，也有一丝希望，因为你遇到了我。你知道赐福吧，就是能让你们褪色者歇息的金光，正是赐福的指引，褪色者该行进的路，指向该去哪里，或者该葬身何处。我想赐福指的是断崖上的城，史东薇尔，又老又丑的半神，接肢葛瑞克的居所，

菈妮
------
- 对话
 + 首次，艾蕾教堂：初次见面啊褪色者，我是魔女蕾娜，听说有褪色者骑着灵马，让我花了点时间去找，看来就是你啊，你能召唤托雷特是吧。我有物品要交给你，是托雷特前主人托给我的，（招魂铃，三匹狼），那个是招魂铃，没有回归黄金树的骨灰，能从中召唤出灵魂，而那些灵魂会暂时认你为主人，重现生前战斗的模样，哎，照你的意思使用吧。褪色者啊，我想我们不会再见面了吧，仔细品尝交接地的滋味吧，我很期待啊，你们褪色者会服从双指到什么时候。

亚人柏克
-------
- 在宁姆格福，关卡前方的一片树丛。
- 对话，初次见面：对哦，我被施法变成树了，我被从洞窟赶出来，回不去了，还被变成树。对不起，我被赶出来时候，东西都忘带了，只能用这个感谢你（蘑菇）。再等我一下，我想偷偷回洞窟，那会东西，我就一定能帮你的忙。但是要再等一下，我还是很怕他们，需要做点心理准备，一想到他们，还有那个海岸的洞窟，我就全身都动不了。
- 对话，在海岸洞窟：你怎么来了，这里很危险，快点逃吧，他们肯定会跑来攻击你，最后落得和我一样的下场。
- 对话，给完缝衣针：这该不会是，缝衣针。谢谢你，我真的很高兴，我妈妈是裁缝，我想成为她那样的裁缝，我不可以这么轻易死掉。

血指猎人-尤拉
--------
- 对话，宁姆格福傍海古遗迹，初见：你是新来的褪色者吧，而且你还成功打到了龙，既然如此，该告诉你，你手上那颗龙心脏，是龙飨用的供品。等到某一天，你为了获得力量吃下它，心中因此萌生渴望，难以克制的时候，记得到西海岸的尽头，有座小岛，到岛上的废弃教堂一趟。不过，你也要记得，所有执行龙飨的人，都会渐渐成为非人，渴望龙的念头不会消失，而是会渐渐积累，最后那个念头充斥全身，不断折磨着你。龙的力量既强大又美丽，却十分可怕，也因此，执行龙飨等同踏上毁灭的不归路。

瑟濂
--------
- 对话，初见，驿站街遗迹：我名为瑟濂，如你所见，是一名魔法师，那么有何贵干？（我想学习魔法），哦你想学习辉石魔法，但老实说，我不认为你能学好辉石魔法，学习魔法并不是只看资质，然而拜师要慎选对象，我被逐出雷亚卢卡利亚学院，是众人避之唯恐不及，所谓的异端魔女，即使如此，你还坚持拜师吗？（我还是想拜师），哈，你真是怪人，好吧，我愿意收你为徒，传授辉石魔法，但我很严格，不走温和教导那一套，可别后悔啊。
- 初次学习魔法，获得动作：点头称是
- 对话，拜师后：徒弟啊，第一步该做的是理解，你在魔法的领域还是个婴儿，极度不成熟，甚至连路都不会走，然而认知到自己的不成熟，才能成为魔法师。我们开始上课吧。
- 对话，拜师后：还想继续上课吗，我乐意奉陪。

肯尼斯海德
---------
- 对话，初次靠近：来人啊，有没有人在，任何能帮助我，肯尼斯海德的人啊，志愿摒除不合理，维护正当秩序的人啊。啊，黄金树，请带来您的侍者吧。
- 对话，初见：哦哦出现了，但是。。。不，即使是褪色者，也值得欢迎，不限身份尊贵与否，皆能维护正当秩序，我是肯尼斯，是宁姆格福领主的嫡长子，褪色者啊，我有事儿要你帮忙。从这里往南，穿过雾林有一个要塞，那是我的要塞，希望你帮我夺回来，都怪史东薇尔派来的骑士长，实在过于驽钝，竟然会为血痴狂。
- 对话，再次：你还在这里做什么，快点往南穿过雾林夺回我的要塞。啊，原来如此，你想问报酬吧，别小看我，打赏我绝不吝啬，只要确实夺回要塞，肯定大大有赏。啊，给你一个建议，最好小心葛瑞克的褪色者猎人，那群家伙一心想抓到你们褪色者，当做恐怖的接肢的祭品。只是凭葛瑞克那种格局，根本不配跻身君王之流。战败之后混在女人堆里逃出王城，忌惮拉塔恩，躲在城里一个气也不敢吭。小看玛莲妮亚，结果输的落花流水，还跪地亲吻她的脚表示服从。实在是软弱无比，不知羞耻的男人。即便出身名门一族，也看得出来葛孚雷的血统淡了很多。仔细想想还真是可悲。
